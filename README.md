## About Swift

**Swift** is a multicurrency banking app built for freelancers, frequent travellers, and B2B. Swift gives you virtual account numbers in **USD, GBP, and EUR**, making international transactions fast, simple, and secure.  

### Key Features

- Open multiple international accounts without traditional banking hassle.  
- Send and receive payments in multiple currencies.  
- Real-time transaction tracking and money management.  
- Sleek, user-friendly mobile experience.  

**Upcoming in Phase 2:**  

- Fingerprint biometrics for ATM cash withdrawals — no need for physical cards.
